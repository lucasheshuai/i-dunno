15. Introduce “Session Design”

A user session should feel satisfying even if short.

Design target

A strong session = 3–5 answered questions

In one session, the user should ideally experience:
one surprising crowd result
one clear demographic split
one profile-building moment
one compelling teaser into the next cluster or next question

Do NOT just randomize everything.
